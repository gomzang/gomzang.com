# Gomzang Brand Assets

These SVG files are tracked as the first working brand system for the site.

- `gomzang-mark.svg`: primary symbol mark for general use.
- `gomzang-small-mark.svg`: simplified optical mark for favicon, avatars, and small UI.
- `gomzang-lockup-horizontal.svg`: horizontal logo lockup.
- `gomzang-lockup-stacked.svg`: stacked logo lockup for square layouts.

The website header uses the symbol mark with live text for better responsiveness. Original PNG exports are kept in `assets/brand/source`.
