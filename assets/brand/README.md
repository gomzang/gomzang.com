# Gomzang Brand Sources

This directory keeps the original raster logo exports for version control.

- `source/gomzang-logo-horizontal.png`: horizontal lockup source.
- `source/gomzang-logo-stacked.png`: stacked lockup source.
- `source/gomzang-mark-primary.png`: primary symbol source.
- `source/gomzang-mark-small.png`: simplified small-size symbol source.

The website itself uses the SVG files in `public/brand` because they scale more cleanly in the header, favicon, and responsive UI.
